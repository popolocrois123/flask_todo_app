

# ToDo List

## 📌 Overview
This project was developed to create a todo list application using Flask.
It focuses on understanding network systems and uses Flask, HTML and jinja2 to create a custom todo list application.

Example:
This project is a web application that allows users to track their daily tasks and improve productivity.

---

## 🎯 Features
- CRUD operations for todo tasks (create, read, update, delete)
- Responsive UI design using HTML and CSS

---

## 🛠 Tech Stack
Frontend: HTML / CSS / Jinja2 / bootstrap
Backend: Flask (Python)
Database: SQLite
Others: Developed on Raspberry Pi with remote access (SSH)
